
# POS Desktop (JavaFX)

Aplicación POS para papelería: versión **desktop** en JavaFX (y más adelante Android).
Este repo sigue buenas prácticas: Maven, JavaFX, FXML + Controller, y Git/GitHub.

## Requisitos
- JDK 24
- Maven (o `mvnw` incluido en el repo)

## Cómo ejecutar (dev)
```bash
mvn javafx:run
